# Crawl-Tiki

Sử dụng Framework Scrapy thu thập dữ liệu Smartphone từ trang Tiki.vn
Các thông tin thu thập được:
- Name
- SKU
- Current_price
- List_price
- Original_price
- Discount
- Discount_rate
- Rating_average
- Review_count
- Favourite_count
- Inventory_status
- Inventory_type
- Day_ago_created
- All_time_quantity_sold
- Brand
- Current_seller
- Rank
- Other_sellers
- Seller_specifications
- Description
